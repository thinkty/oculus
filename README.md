# Oculus: A simple screenshot application made with Qt

## Plans
### Screenshot
- Take a screenshot of a selected area or fullscreen
- By default, it will be saved in the project directory with Screenshot_YYYY-MM-DD_HH-MM-SS.png
- Add an option to upload it to Imgur

### Screen Record
- Record the selected area or fullscreen
- Set a timer to start recording
- Save the recorded file in MP4 or GIF
- Save the file locally or upload it to Imgur
